# Rollen

Basically tinder but for movies

## How to deploy

```sh
git clone git@github.com:grant/rollen.git;
cd rollen;
npm install;
```

Go to `0.0.0.0:3000`

## How to develop

### Install

```sh
npm install gulp -g
```

### Run

```sh
gulp
```
